# How to Add Drop Shipping Products to Your Shopify Store

## Quick Start Guide

### Step 1: Access Shopify Admin
1. Go to your Shopify admin: `https://by2050-test.myshopify.com/admin`
2. Log in with your credentials

### Step 2: Add Products Manually

**Option A: Single Product**
1. Click **Products** in the left sidebar
2. Click **Add product**
3. Fill in:
   - **Title**: Product name
   - **Description**: Product details
   - **Media**: Upload product images
   - **Pricing**: Set price and compare at price
   - **Inventory**: Set SKU, barcode, quantity
   - **Shipping**: Weight and dimensions
   - **Variants**: Size, color, etc.
4. Click **Save**

**Option B: Import Products (CSV)**
1. Go to **Products** → **Import**
2. Download the sample CSV template
3. Fill in your product data
4. Upload the CSV file
5. Review and import

### Step 3: Use Drop Shipping Apps

**Recommended Apps:**
1. **Oberlo** (by Shopify)
   - Connect to AliExpress
   - Import products with one click
   - Auto-fulfill orders

2. **DSers** (AliExpress Dropshipping)
   - Bulk import products
   - Auto-order fulfillment
   - Inventory sync

3. **Spocket**
   - US/EU suppliers
   - Faster shipping
   - Higher quality products

**How to Install:**
1. Go to **Apps** in Shopify admin
2. Click **Visit Shopify App Store**
3. Search for the app (e.g., "Oberlo")
4. Click **Add app**
5. Follow setup instructions

### Step 4: Connect Products to Collections

1. Go to **Products** → Select a product
2. Scroll to **Collections**
3. Add to existing collection or create new
4. Collections appear on your homepage sections

### Step 5: Set Up Collections for Homepage

1. Go to **Products** → **Collections**
2. Create collections like:
   - "New Arrivals"
   - "Best Sellers"
   - "Sustainable Collection"
3. Add products to collections
4. Go to **Online Store** → **Themes** → **Customize**
5. In homepage sections, select your collections

## Product Requirements for Your Theme

### Required Fields:
- **Title**: Product name
- **Price**: Selling price
- **Images**: At least 1 product image (recommended: 2-4 images)
- **Description**: Product details

### Recommended Fields:
- **Compare at price**: For sale badges
- **Variants**: Size, color options
- **Tags**: For filtering (e.g., "sustainable", "organic")
- **Metafields**: For sustainability metrics

## Sustainability Features

Your theme supports:
- **Sustainability badges**: Add tags like "eco-friendly", "organic"
- **Sustainability metrics**: Use metafields for CO₂ saved, water saved
- **Material tags**: Tag products with materials (e.g., "organic cotton")

## Quick Tips

1. **Product Images**: Use high-quality images (minimum 800x800px)
2. **Descriptions**: Write detailed, SEO-friendly descriptions
3. **Pricing**: Set competitive prices with compare-at prices for sales
4. **Inventory**: Keep track of stock levels
5. **SEO**: Fill in product page title and description meta fields
