## Next Steps (Terminal)

### 1. Authenticate (First Time Only)

Open a **new terminal** (PowerShell or Command Prompt) and run:

```bash
claude auth login
```

**You'll need:**
- Anthropic account (Pro or Max plan - $20+/month)
- Visit https://claude.ai to sign up if needed
- Follow the authentication prompts

### 2. Test It Works

```bash
claude --version
# Should show: 2.0.76 (Claude Code)
```

### 3. Try It Out

```bash
# Ask a question
claude "What is this project about?"

# Run health check
claude healthcheck

# Interactive mode
claude
```
